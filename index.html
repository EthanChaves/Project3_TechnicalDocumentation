Syntax

Here's an example styling an element ( later in this article we are going to explain every single thing in this example )

.class {
background-color: #fff;
 margin: 10px auto;
 padding: 0;
 text-align: center;
 }

As we can see in the box right before, is simple, firstable there's a selector and inside of the parenthesis ( { } ) will be all the changes that we are going to make in the element selected.

Selectors

A selector is the part that selects the element in the html document that we are going to style, there are different selectors:

Selector Use
* Selects every single element in the document (general selector)
.class Selects the elements with the specified class, you need to use a period and the name of the class after that
#id Selects the element with the specified id, you need to use the numeral symbol and the name of the id after that
h1, body, header Selects an specific type of element, if you separate every element with a coma, you can select many different elements that need the same changes
div.class This selects all the elements with that class, but only the ones that are inside of a div (elements with the class that are not contained inside of a div won't be affected)
span [class="class"] This is other way to select a class inside an element
Colors

There are 4 different ways to select an specific color:

  • Pseudo: There are some basic colors that are preset, for example: black, white, green, blue, etc.
  • RGB: This uses a different syntax rgb(#, #, #). Every number inside of the parenthesis stands for the amount of color, red for the first one, green for the second and blue for the last one. This numbers can be in a range between 0 and 255.
  • RGBA: This one uses a similar syntax rgba(#, #, #, #). The first 3 numbers stands for the same as in the rgb style but this one has a fouth number that stands for the opacity.
  • Hexadecimal: This is the last way to select a color, this uses a more simple syntax #000000. As it's name said, this uses an hexadecimal syntax (in a range from 0 to f) but using the same idea as before, the first 2 digits stands for the red, the second 2 digits for the green and the last ones for blue.
