Syntax

Here's an example styling an element ( later in this article we are going to explain every single thing in this example )

.class {
background-color: #fff;
 margin: 10px auto;
 padding: 0;
 text-align: center;
 }

As we can see in the box right before, is simple, firstable there's a selector and inside of the parenthesis ( { } ) will be all the changes that we are going to make in the element selected.

Selectors

A selector is the part that selects the element in the html document that we are going to style, there are different selectors:

Selector Use
* Selects every single element in the document (general selector)
.class Selects the elements with the specified class, you need to use a period and the name of the class after that
#id Selects the element with the specified id, you need to use the numeral symbol and the name of the id after that
h1, body, header Selects an specific type of element, if you separate every element with a coma, you can select many different elements that need the same changes
div.class This selects all the elements with that class, but only the ones that are inside of a div (elements with the class that are not contained inside of a div won't be affected)
span [class="class"] This is other way to select a class inside an element
Colors

There are different ways to select an specific color:

  • Pseudo: There are some basic colors that are preset, for example: black, white, green, blue, etc.
  • RGB: This uses a different syntax rgb(#, #, #). Every number inside of the parenthesis stands for the amount of color, red for the first one, green for the second and blue for the last one. This numbers can be in a range between 0 and 255.
  • RGBA: This one uses a similar syntax rgba(#, #, #, #). The first 3 numbers stands for the same as in the rgb style but this one has a fouth number that stands for the opacity.
  • HSL: The syntax in this one is hsl(hue, saturation, lightness). The first is a normal digit from 0 to 360 as in the color wheel (0 is red, 120 green and 360 blue), and the other two elements are percentages from 0% to 100%
  • Hexadecimal: This is the last way to select a color, this uses a more simple syntax #000000. As it's name said, this uses an hexadecimal syntax (in a range from 0 to f) but using the same idea as before, the first 2 digits stands for the red, the second 2 digits for the green and the last ones for blue.

span {
background-color: red;
 }

h1 {
color: rgb(255, 221, 0);
 }

#navbar {
background-color: #002e57;
 }

Styling Elements

Background:

You can style the background in every single element:

  • background-color: Changes the color of the background using any of the color selection styles.
  • background-image: For this one you should use the url of an image (just use url("www.exampleurl.com")) or if you have the image downloaded in your device, you can select it by its direction in your folders.
  • background-repeat: If you want your background image to repeat in any axis,or not, you can select this property (some images depending of it's dimensions can be repeated by default)
  • background-position: This property is used to select the position of the image in the background.
  • background: This is a shorthand property that wrap all the properties before in a single one, it recieve first the color for the background, later the image url (if you want to), if you want your image to repeat and the last two values are the position of the background image.

section {
background-image: ./bricks.png;
 }

ul {
background: #fff url("www.exampleurl.com") no-repeat right bottom;
 }

Border:

You can change the style of your element's border or make it invicible if you want.

  • border-color: Changes the color of the border using any of the color selection styles.
  • border-style: You can change the form of your border, it could be solid, dotted, double, etc.
  • broder-radius: This will make the border corners rounded, you can select how rounded the corners will be.
  • border-width: This will change the thickness of the border.
  • border-top / border-bottom / border-right / border-left: This properties will make the same changes as before but only in the border at the selected side of the element.
  • border: This is a shorthand property that wrap all the properties before in a single one, it's syntax is simplier, this property recieves a width, a color and the style of the border.

div {
border: 5px blue double;
 border-radius: 10px;
 }

Outline:

Is the line between the border and the margin.

  • outline-color: Changes the color of the line.
  • outline-style: Same as in the border, it refers to the form of it, it could be solid, dotted, etc.
  • outline-width: This changes the thickness of the line.
  • outline: This shorthand property wrap all the properties mentioned before. it's syntax is the same as the border property, first comes the width, later the color and and at the end the style of the line.

section {
outline: 3px #faffaa solid;
 }

Text and font:

You can style your text in different ways:

  • text-align: Aligns the text to the place that you want it to be (uses the parent element as reference)
  • text-shadow: This will make a shadow behind the text, it will recieve 2 values to ubicate it (hotizontal and vertical) and a third one that will selects it's color.
  • text-decoration: This property will decorate the text. It may recieve 4 different values, only the first one (where the line goes) is required, the other ones are opcional.
  • font-family: With this one you can select one or more different fonts to your text (in case that the first one is not supported by browser, the next one will be selected.
  • font-style: It changes the style of your text, for example if you want it italic, oblique...
  • font-weight: Changes how bold or thin text letters will be.
  • font-size: With this property you can change the size of text.

h3 {
font-family: sans-serif;
 text-decoration: underline blue;
 text-align: center;
 }

Links status:

You can style your link elements (< a >) depending of the status in a different moment:

  • a:link is the style of your link when it is unvisited.
  • a:visited is the style when you already visited the link.
  • a:hover is the style when you put your mouse over the link.
  • a:active is the style when you click the link.

a:link {
color: white;
 background-color: blue;
 border: 3px solid white;
 }

a:visited {
color: yellow;
 background-color: blue;
 border: 3px solid yellow;
 }

a:hover {
color: red;
 background-color: white;
 border: 3px solid red;
 }

NOTE: The order that selectors appeared in the list before is important because if you don't follow this order, the styles aren't going to work.

Box-model and Position

Box Model:

Here is an ilustration to simplify what we are going to learn in this part:

Every element may recieve one, two or four values. If it takes one value, it will me applied to every side of the element. If it takes two values, the first one will be applied for the top and bottom and the second one will be applied to the right and left sides. And if it takes four values, every value will be applied in this order: top, right, bottom and left.

  • margin: Is the space outside the element.
  • padding: Is the space between the border and the content of the element.
  • width and height: This are the dimensions for the element.
  • display: This changes how an element is displayed, for example, if you give it the value of 'inline' the content will be display all in the same line, if you use the 'block' value elements will be display one right below of the one before.
  • position: This property changes how an element will be positionated, for example, if you use the value 'fixed' the element will be in the position that you gave it, doesn't matter if you scroll in the page, if you use the value 'relative' this will let you positionate your element wherever you want.
    When you use this property, you can use the other properties top, right, left and bottom to move your element.

.box {
width: 50%;
 margin: 25px;
 }

#image {
position: relative;
 left: -20px;
 top: 0;
 }
